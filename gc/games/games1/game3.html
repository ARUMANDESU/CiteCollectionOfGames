Minecraft

Genre: Action, Simulation, Adventure, Puzzle
Platform: PC / PlayStation 3 / PlayStation 4 / PlayStation 5 / Xbox 360 / Xbox One / Xbox Series / Nintendo Switch / Android / iOS / PlayStation Vita
Release date: 2011.11.18
Publisher: Microsoft
The developer: 4J Studios / Markus Persson / Mojang

Minecraft is an indie open-world sandbox game with survival
Grand Theft Auto series, made in the Action / Sandbox genre. For the
elements. In style, the game world is entirely made up of blocks
(terrain, items, mobs, player), and low resolution textures (16x16
texels) are used for texturing. The game was originally developed by
Swedish programmer Markus Persson, also known as "Notch". The
game was conceived as a clone of Infiniminer, although Persson
expressed a desire to simulate its gameplay as Dwarf Fortress. The
game is written in Java using the LWJGL library.

The gameplay in the game is simple - players extract resources to
build houses, castles and entire cities. There are practically no
restrictions in Minecraft, except for the height of the level - otherwise,
players are free to create whatever they want.

In addition to the computer version, Minecraft also has special mobile
versions for smartphones, a version for the PS Vita, and a version for
consoles, each in its own way. which differs from the standard
version.
